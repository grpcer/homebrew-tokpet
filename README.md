# homebrew-tokpet

Homebrew tap for [tokpet](https://github.com/grpcer/tokpet) — a desktop pet that
surfaces your real-time AI usage / quota / balance across providers.

## Install

```bash
brew install grpcer/tokpet/tokpet
brew services start tokpet   # runs in the background, restarts on login
tokpet open                  # configure providers in your browser
```
